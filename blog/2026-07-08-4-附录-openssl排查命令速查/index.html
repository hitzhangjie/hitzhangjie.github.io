Etcd证书有效期延长带来的思考(4)：附录 OpenSSL 排查命令速查

Posted 2026-07-08 12:00 +0800 by ZhangJie ‐ 2 min read

分享:  

附录：OpenSSL 排查命令速查 #

使用说明 ##

  • 目的：快速定位“文件内容是否正确、证书与私钥是否配对、证书是否由指定 CA 签发”。
  • 建议：先看核心字段，再做配对和验签，最后结合业务程序行为确认。

1) key 文件（私钥）查看 ##

# 查看私钥基本信息（兼容 RSA/ECDSA）
openssl pkey -in etcd-key.pem -text -noout

# 导出公钥（用于和证书公钥比对）
openssl pkey -in etcd-key.pem -pubout -outform pem

重点关注：

  • 算法类型（RSA / EC）
  • 位数（如 2048/4096）
  • 是否可正常导出公钥

2) CA 证书查看 ##

# 查看 Subject / Issuer / 有效期 / 序列号
openssl x509 -in ca.pem -noout -subject -issuer -dates -serial

# 查看 CA 关键扩展
openssl x509 -in ca.pem -noout -text | rg "Basic Constraints|Key Usage|Subject Key Identifier|Authority Key Identifier" -n

重点关注：

  • Basic Constraints 是否含 CA:TRUE
  • Key Usage 是否含 Certificate Sign
  • 到期时间是否满足轮换窗口

3) 普通证书查看 ##

# 查看 Subject / Issuer / 有效期 / 序列号
openssl x509 -in etcd.pem -noout -subject -issuer -dates -serial

# 查看 SAN
openssl x509 -in etcd.pem -noout -ext subjectAltName

# 查看用途与 CA 属性
openssl x509 -in etcd.pem -noout -text | rg "Key Usage|Extended Key Usage|Basic Constraints" -n

重点关注：

  • Basic Constraints 是否为 CA:FALSE
  • EKU 是否包含 TLS Web Server Authentication / TLS Web Client Authentication
  • SAN 是否覆盖实际访问 DNS/IP（尤其 LB 地址）

4) 证书与私钥配对检查 ##

# 私钥公钥摘要
openssl pkey -in etcd-key.pem -pubout -outform pem | openssl sha256

# 证书公钥摘要
openssl x509 -in etcd.pem -pubkey -noout | openssl sha256

判定规则：

  • 两个摘要一致 => 证书与私钥配对
  • 不一致 => 文件拿错或证书/私钥不匹配

5) 指定 CA 验签检查 ##

openssl verify -CAfile new-ca.pem etcd.pem

判定规则：

  • OK => 在该工具判定下通过
  • 报错 => 先检查链路与 DN，再结合业务程序（Go crypto/x509 / etcd 实际行为）确认

6) 边界场景提醒 ##

  • 在同 DN 等边界场景中，openssl verify 可能出现误判。
  • 不要只看工具输出，要结合程序实际建链和验签结果一起判断。
